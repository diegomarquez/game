Additional Game-Builder Tools
A little something to try to make development easier. You will need node and git, so go install them!

Generator Game-Builder

A Yeoman generator to scaffold a Game-Builder project. https://github.com/diegomarquez/generator-game-builder

  1. Open your favorite shell
  2. npm install yo -g
    • This will make the yo command globally available.
  3. git clone
    • https://github.com/diegomarquez/generator-game-builder
  4. cd generator-game-builder
  5. npm link
    • This will make game-builder a valid argument for yo.
  6. mkdir my_cool_project
  7. cd my_cool_project
  8. yo game-builder
    • This will get all the stuff needed to get started with Game-Builder.
  9. Additional information on the generator
    • https://github.com/diegomarquez/generator-game-builder
That's all the extra help provided by Game-Builder. Let's hope it is enough.
