A requireJS module. For use with Game-Builder

This is used in game-object to call the debugdraw method of all the components. It also draws the registration point of the the game-object calling it's debugdraw method.

debugDraw

Draw info about a game-object.

This method only does something if the debug property in gb is set to true and the property skipDebug of the game-object making the call is set to false.

Parameters:

  • context must be a Context 2D.
    (CanvasRenderingContext2D)

  • viewport must be an Object.
    (The viewport the game-object is drawn too)
