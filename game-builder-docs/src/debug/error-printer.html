A requireJS module. For use with Game-Builder

This module is used all around Game-Builder to print out errors to console.

In the spirit of requireJS, the error has a link to a more detailed explanation on what happened.

Error Printing

define(function() { var ErrorPrinter = function() {} var getAnchor = function(origin) { return '#' + origin.toLowerCase() .trim() .replace(/ /g, '_'); };

printError

Prints a nice error to console and link to the corresponding place in Game-Builder website for a more detailed explanation.

Parameters:

  • origin must be a String.
    (An id to determine from which place this method was called.)

  • message must be a String.
    (Error message)

  • e is optional, must be an Object, and has a default value of null.
    (Optional Error argument, it is thrown if sent.)

Can throw an Error
(Always)

ErrorPrinter.prototype.printError = function(origin, message, e) { throw new Error(origin + ' => ' + message + '\nhttp://diegomarquez.github.io/game-builder/errors.html' + getAnchor(origin)); if (e) { throw e; } };
