A requireJS module. For use with Game-Builder

This module defines the object which is responsible for the main update loop of the application.

It does so by using request animation frame

It also hooks into the focus and blur events of window, in order to be able to pause and resume the application if needed.

It's main behaviour can be extended by the use of extensions

Game extends delegate so it provides a few events to hook into:

create

On application startup 

game.on(game.CREATE, function(repeatsLeft) {});

blur

When the application looses focus 

game.on(game.BLUR, function() {});

focus

When the application gains focus

game.on(game.FOCUS, function() {});

update

On each update cycle

The delta time between update cycles is sent to registered callbacks

game.on(game.UPDATE, function(delta) {});

create

The main method that will kick start everything.

This method does a bunch of things. The main one being setting up the update loop using request animation frame

The focus and blur events are also setup here.

If the application does not have focus as soon as it starts, it waits until it has focus to setup the update loop.

Parameters:

  • mainContainer must be an ElementObject.
    (The Dom element that contains the canvas)

  • canvas must be a Canvas.
    (A Canvas object)

The blur and focus event callbacks Plus additional logic so that they don't get executed more than needed.

var blur = false; var onBlur = function() { var oldBlur = self.blur; if (self.blur) { self.blur = false; self.focus = true; if (!blur) { self.execute_extensions(this.BLUR); self.execute(this.BLUR); blur = true; } } return oldBlur; }; var onFocus = function() { var oldFocus = self.focus; //In the case the game is not already created when the document gains focus //for the first time, it is created here. if (!self.initialized) { setupUpdateLoop(); } else { if (self.focus) { self.blur = true; self.focus = false; if (blur) { self.execute_extensions(this.FOCUS); self.execute(this.FOCUS); blur = false; } } } return oldFocus; }
