A requireJS module. For use with Game-Builder

This module defines an extension that uses layers to halt all update activity when the application looses focus.

The extension also adds a pause method to game to be able to pause the application manually.

This Extension adds an event game can hook into:

pause

When the application is paused manually

game.on(game.PAUSE, function() {});

Pause activity
