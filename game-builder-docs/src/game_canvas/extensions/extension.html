A requireJS module. For use with Game-Builder

This module only exists to make the interface for an extension explicit.

Extensions works with game. They provide functionality that is common enough to be executed by game, but not so common to make it part of the core of Game-Builder.

Extensions can be hooked into 4 parts of game:

Initialization

These extensions return the string 'create' for their type. They are executed once on the initialization of the application

Focus

These extensions return the string 'focus' for their type. They are executed each time the application gains focus.

Blur

These extensions return the string 'blur' for their type. They are executed each time the application looses focus.

Update

These extensions return the string 'update' for their type. They are executed on the main update loop.
