A requireJS module. For use with Game-Builder

This module defines an extension that uses layers to resume all update activity when the application gains focus.

The extension also adds a resume method to game, to be able to resume the application manually after pausing.

This Extension adds an event game can hook into:

  • ### resume When the application is resumed manually
game.on(game.RESUME, function() {});
