A requireJS module. For use with Game-Builder

This module defines a component meant to be attached to a game-object, to give it the ability to collide against other game-object with collider components.

Fixed Polygon colliders are just like a polygon-collider, the difference is that they do no follow their parents along all the transformations, only translation.

This make them considerably less expensive, but also less accurate. Often you can do without the precision.

During the configuration of the component-pool fixed polygon colliders need to receive an object that looks similar to the following:

gb.coPool.createConfiguration("Fixed_Polygon", 'Fixed_Polygon_1')
    .args({
        //Id used by the Collision Resolver
        id:'fixed-polygon-collider_ID', 

        //Array of points that define the polygon collider
        points:[ 
            new vector_2D(-10, -10), 
            new vector_2D(10, -10), 
            new vector_2D(10, 10), 
            new vector_2D(-10, 10) 
        ]
});

If it is not provided it will most likely fail in un-expected ways.

Note: The snippet uses the reference to the component-pool found in the gb module. The way you get a hold to a reference to the component-pool may vary.

define(['collision-component', 'sat', 'collision-resolver', 'vector-2D', 'draw'], function(CollisionComponent, SAT, CollisionResolver, Vector2D, Draw) { var p = {}; var m = null; var Component = CollisionComponent.extend({

start

Set up the collider.

Creates a FixedSizePolygon object defined in the sat module. The polygon is said to be fixed of size becuase no more vertexes can be added to it after it is created.

start: function() { this._super(); this.pointCount = this.points.length; this.pointsCopy = JSON.parse(JSON.stringify(this.points)); this.collider = new SAT.FixedSizePolygon(new Vector2D(0, 0), this.points); this.colliderType = CollisionResolver.polygonCollider; },

update

Updates the position of the collider.

The collider follows the position of it's parent.

update: function() { this.parent.getTransform(p, m); this.collider.pos.x = p.x; this.collider.pos.y = p.y; this._super(); },

debug_draw

Draw the fixed polygon collider

This method is only executed if the debug property of the parent gb is set to true. It is better to leave the drawing to the renderer components.

Parameters:

  • context must be a Context 2D.
    (Canvas 2D context)

  • viewport must be an Object.
    (A reference to the current viewport)

  • draw must be an Object.
    (A reference to the draw module)

debug_draw: function(context, viewport, draw) { this.parent.getTransform(p, m); context.save(); context.setTransform(1, 0, 0, 1, 0, 0); context.translate(p.x, p.y); Draw.polygon(context, 0, 0, this.pointsCopy, null, this.debugColor, 2); context.restore(); this._super(); }
