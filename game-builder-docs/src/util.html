shallow_merge

Performs a shallow merge of two objects.

It is shallow because reference values are kept as references. It creates a new object into which all the properties of the first object are copied into, then all the properties of the second object are copied into it.

This means that properties of the second object have precedence over those in the first object.

Parameters:

  • first must be an Object.
    (The first object)

  • second must be an Object.
    (The second object)

Returns an Object
(Result of merging)

Utils.prototype.shallow_merge = function(first, second) { var result = {}; this.shallow_copy(first, result); this.shallow_copy(second, result); return result; };

shallow_merge_many

Performs a shallow merge of many objects.

The object which are specified first will be overriden by the ones specified last

Returns an Object
(Result of merging)

Utils.prototype.shallow_merge_many = function() { var result = {}; for (var i = 0; i < arguments.length; i++) { if (arguments[i]) { this.shallow_copy(arguments[i], result); } } return result; };

shallow_copy

Performs a shallow copy properties.

The copy is shallow because reference values are kept as references.

Parameters:

  • from is optional, must be an Object, and has a default value of {}.
    (The Object from which to copy properties)

  • to is optional, must be an Object, and has a default value of {}.
    (The Object to copy into)

Returns an Object
(Result of the copying)

Utils.prototype.shallow_copy = function(from, to) { from = from || {}; to = to || {}; for(var k in from) { to[k] = from[k]; } return to; };

iterateObject

Iterates through the properties of an object, executing a callback for each of them

Parameters:

  • o must be an Object.
    (Object to iterate )

  • c must be a Function.
    (Callback to execute on each property of the provided object )

Utils.prototype.iterateObject = function(o, c) { for(var propName in o) { if(o.hasOwnProperty(propName)){ if (o[propName]) { c(o, propName); } } } };

destroyArray

Null all the elements of an array.

Parameters:

  • a must be an Array.
    (Array to destroy)

  • resetLength is optional, must be a Boolean, and has a default value of false.

Utils.prototype.destroyArray = function(a, resetLength) { for(var i=0; i<a.length; i++) { a[i] = null; } if(resetLength) { a.length = 0; } };

cache

Get a function that will cache the first computed value

Parameters:

  • func must be a Function.
    (Function to cache)

  • scope can be an Oject or a null.
    (Scope to bind to)

  • args can be an Object or a null.
    (Arguments the cached function will receive)

Returns a Function
(Function that returns the first computed value. When called, this function may recieve a Boolean argument to force a refresh, by default it is false.)

Utils.prototype.cache = function(func, scope, args) { var r; var s; var a; s = scope || this; a = args; return function (refresh) { if (!r) { r = func.apply(s, a); } if (refresh) { r = func.apply(s, a); } return r; } }
