A requireJS module. For use with Game-Builder

It implements a Multicast Delegate. Sounds like a mouthful? The more friendly name is, 'Event System', which sounds about a million times less cool.

Now you know what the hell everyone is trying to talk about when they mention events.

Basically this class is a hash, with each key of the hash being an array of functions.

Add funtions using on, always providing an id and scope with the function. At some point in the future call the method execute, passing an id. All the functions registered under the id provided will be executed in the order they were added.

Simple, flexible and powerful.

on Use it to add functions to the delegate instance.

Parameters:

  • name must be a String.
    (Id that the function will be associated with)

  • scope must be an Object.
    (Scope of the function, most of the time you will be passing 'this')

  • callback must be a Function.
    (Function you want to execute)

  • removeOnExecute is optional, must be a Boolean, and has a default value of false.
    (The function will be removed from the corresponding list, after executing it once)

  • inmediate is optional, must be a Boolean, and has a default value of false.
    (Execute function inmediatelly after adding it)

  • keepOnCleanUp is optional, must be a Boolean, and has a default value of false.
    (Save the function when executing the softCleanUp)

  • single is optional, must be a Boolean, and has a default value of false.
    (Do not add function if there is already one with the same id)

Returns a null

on: function(name, scope, callback, removeOnExecute, inmediate, keepOnCleanUp, single) { if (!this.callbackList[name]) { this.callbackList[name] = []; } if (inmediate) { callback(); } if (single) { if (this.callbackList[name].length == 1) { return; } } this.callbackList[name].push({ scope: scope, callback: callback, removeOnExecute: removeOnExecute, keep: keepOnCleanUp }); },

remove Removes the specified function from the array it is in.

Parameters:

  • name must be a String.
    (Id the funtion you want to remove is associated with)

  • scope must be an Object.
    (Scope used when adding the function to the delegate)

  • callback must be a Function.
    (Function you want to remove from the delegate)

Returns a null

remove: function(name, scope, callback) { this.list = this.callbackList[name]; if (!this.list) return; for (var i = this.list.length - 1; i >= 0; i--) { var callbackObject = this.list[i]; if (scope === callbackObject.scope && callback === callbackObject.callback) { this.list.splice(i, 1); } } },

removeAll Removes all the funtions associated with an id.

Parameters:

  • name must be a String.
    (All funtions matching this Id will be removed from the delegate)

Returns a null

removeAll: function(name) { var list = this.callbackList[name]; if (list) { list.splice(0, list.lenght); list.lenght = 0; list = null; } },

softCleanUp Removes every function in the delegate, except for the ones that were configured to be kept in on.

Returns a null

softCleanUp: function() { for (var k in this.callbackList) { this.list = this.callbackList[k]; if (!this.list) return; for (var i = this.list.length - 1; i >= 0; i--) { var callbackObject = this.list[i]; if (!callbackObject.keep) { this.list.splice(i, 1); } } } },

hardCleanUp Removes every function in the delegate.

Returns a null

hardCleanUp: function() { for (var k in this.callbackList) { this.removeAll(k); } },

Returns a null

execute Use this to call all the methods registered using on.

Parameters:

  • name must be a String.
    (All the funtions registered with the id provided will be executed)

  • args must be an Object.
    (This Object will be passed as argument to all the funtions executed)

Returns a null
