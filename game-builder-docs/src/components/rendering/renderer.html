A requireJS module. For use with Game-Builder

This module only exists to make the interface for a renderer explicit.

An Interface for renderers

init

Constructor

init: function() { this._super(); this.offsetX = 0; this.offsetY = 0; this.scaleX = 1; this.scaleY = 1; this.debugColor = "#FFFF00"; },

draw

This is an abstract method and must be overriden.

Can throw an Error
(Always)

draw: function(context) { ErrorPrinter.mustOverrideError('Renderer'); },

rendererWidth

Returns a Number
(The width of the renderer)

rendererWidth: function() { return this.width * this.scaleX; },

rendererHeight

Returns a Number
(The height of the renderer)

rendererHeight: function() { return this.height * this.scaleY; },

rendererOffsetX

Returns a Number
(The offset in the X axis of the renderer)

rendererOffsetX: function() { return this.offsetX * this.scaleX; },

rendererOffsetY

Returns a Number
(The offset in the Y axis of the renderer)

rendererOffsetY: function() { return this.offsetY * this.scaleY; },

debug_draw

This method is only executed if the debug property of the parent gb is set to true. It is better to leave the drawing to the renderer components.

Parameters:

  • context must be a Context 2D.
    (Canvas 2D context)

  • viewport must be an Object.
    (A reference to the current viewport)

  • draw must be an Object.
    (A reference to the draw module)
