A requireJS module. For use with Game-Builder

This module defines an object to contain and work with layer objects.

It also acts as a wrapper for interacting with root.

Not much else needs to be said about this module, it's not very complex.

Layer management

define(["root", "layer", "error-printer"], function(Root, Layer, ErrorPrinter) { var LayerContainer = function() { this.layers = {}; };

add

Adds a new layer. The layer is managed in this object, but it also is added to root, which is what will actually make it display.

On that subject, it is important to not that if you re-arrange the elements of the array that the layer is added to, nothing will happen to the order of updating or rendering, as that is controlled by the root, which has a list of childs of it's own.

Parameters:

  • name must be a String.
    (Id of the layer, used later to refer to the layer.)
LayerContainer.prototype.add = function(name) { var layer = new Layer(); Root.add(layer).start(); this.layers[name] = layer; return layer; };

remove

Removes a layer from the array in this object, and also from the root. Prior to removing the layer it also clears it from any game-objects

A removed layer can not be used again, if you wish to just remove all game-objects from a layer you might want to use the clear method.

Parameters:

  • name must be a String.
    (Id of the layer to remove)
LayerContainer.prototype.remove = function(name) { this.layers[name].clear(); delete this.layers[name]; Root.remove(this.layers[name]); };

clear

Clears a layer from all game-objects

A cleared layer can still be used to add more things to it.

LayerContainer.prototype.clear = function(name) { this.layers[name].clear(); };

get

Get a reference to a layer

Parameters:

  • name must be a String.
    (Id of the layer to get a reference to.)

Can throw an Error
(If the provided name does not match any layer)
and Returns a Layer
(The specified layer. )

LayerContainer.prototype.get = function(name) { if (!this.layers[name]) { ErrorPrinter.printError('Layers', 'Layer with id:' + name + ' does not exist.') } return this.layers[name]; };

stop

Stops updating and rendering everythin in the requested layer. For practical purposes this method pauses and makes invisible all the contents of a layer.

Parameters:

  • name must be a String.
    (Id of the layer in which to stop all activity)
LayerContainer.prototype.stop = function(name) { this.stop_update(name); this.stop_draw(name); };

resume

Resumes all activity in the specified layer.

Parameters:

  • name must be a String.
    (Id of the layer to resume activity in )
LayerContainer.prototype.resume = function(name) { this.resume_update(name); this.resume_draw(name); };

stop_draw

Stops rendering of the specified layer. Effectively making everything in it invisible.

Parameters:

  • name must be a String.
    (Id of the layer that should stop rendering)
LayerContainer.prototype.stop_draw = function(name) { this.layers[name].canDraw = false; };

resume_draw

Resumes rendering of the specified layer.

Parameters:

  • name must be a String.
    (Id of the layer that should resume rendering)
LayerContainer.prototype.resume_draw = function(name) { this.layers[name].canDraw = true; };

stop_update

Stops updating of the specified layer. Effectively pausing everything in it.

Parameters:

  • name must be a String.
    (Id of the layer that should stop updating)
LayerContainer.prototype.stop_update = function(name) { this.layers[name].canUpdate = false; };

resume_update

Resumes updating of the specified layer.

Parameters:

  • name must be a String.
    (Id of the layer that should resume updating)
LayerContainer.prototype.resume_update = function(name) { this.layers[name].canUpdate = true; };

all

Performs a given action on all the layers. The two arguments are concatenated to form the name of one of the methods described earlier to control layers. ej.:

  • javascript layers.all('stop', 'update');

Would stop the updating of all the layers.

Parameters:

  • action must be a String.
    (This can be either 'resume' or 'stop')

  • method must be a String.
    (This can be either 'draw' or update.)

LayerContainer.prototype.all = function(action, method) { if (method) action = action + '_' + method; for (var k in this.layers) { this[action](k); } };
