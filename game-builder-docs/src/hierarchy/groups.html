Inherits from: delegate

Depends of: root group error-printer

A requireJS module. For use with Game-Builder

This module defines an object to contain and work with group objects.

It also acts as a wrapper for interacting with root.

This object extends delegate so it provides a few events to hook into:

ADD

When a group is added

Registered callbacks get the group as argument 

gb.groups.on(gb.groups.ADD, function(group) {});


REMOVE

When a group is removed

Registered callbacks get the group as argument 

gb.groups.on(gb.groups.REMOVE, function(group) {});


CHANGE

When group changes position

Registered callbacks get the group as argument 

gb.groups.on(gb.groups.CHANGE, function(group) {});


REMOVE_ALL

When all the groups are removed

gb.groups.on(gb.groups.REMOVE_ALL, function() {});

Note: The snippets use the reference to groups found in the gb module. The way you get a hold to a reference to groups may vary.

Group management

define(["delegate", "root", "group", "error-printer"], function(Delegate, Root, Group, ErrorPrinter) { var GroupContainer = Delegate.extend({

init

Constructor

init: function() { this._super(); this.groups = {}; this.groupsArray = []; },

add

Adds a new group. The group is managed in this object, but it also is added to root, which is what will make it update.

On that subject, it is important to note that if you re-arrange the elements of the array that the group is added to, nothing will happen to the order of updating or rendering, as that is controlled by the root, which has a list of childs of it's own.

Parameters:

  • name must be a String.
    (Id of the group, used later to refer to the group.)

Returns an Object
(The group that was just created)

add: function(name) { var group = new Group(name); Root.add(group).start(); this.groups[name] = group; this.groupsArray.push(group); this.execute(this.ADD, group); return group; },

remove

Removes a group from the array in this object, and also from the root. Prior to removing the group it also clears it from any game-objects

A removed group can not be used again, if you wish to just remove all game-objects from a group you might want to use the clear method.

Parameters:

  • name must be a String.
    (Id of the group to remove)
remove: function(name) { var group = this.groups[name]; this.execute(this.REMOVE, group); group.clear(); this.groupsArray.splice(this.groupsArray.indexOf(group), 1); delete this.groups[name]; Root.remove(this.groups[name]); },

removeAll

Removes all groups from the array in this object, and also from the root.

removeAll: function() { for (var k in this.groups) { this.remove(k); } this.execute(this.REMOVE_ALL); },

clear

Clears a group from all game-objects

A cleared group can still be used to add more things to it.

clear: function(name) { this.groups[name].clear(); },

get

Get a reference to a group

Parameters:

  • name must be a String.
    (Id of the group to get a reference to.)

Can throw an Error
(If the provided name does not match any group)
and Returns a Group
(The specified group.)

get: function(name) { if (!this.groups[name]) { ErrorPrinter.printError('Groups', 'group with id:' + name + ' does not exist.') } return this.groups[name]; },

exists

Tell whether a group exists or not

Parameters:

  • name must be a String.
    (Id of the group to check existance of.)
exists: function(name) { return this.groups[name] ? true : false; },

change

Change the position of the specified group

Parameters:

  • name must be a String.
    (Id of the group to arrange.)
change: function(name, index) { var group = this.groups[name]; var groupIndex = this.groupsArray.indexOf(group); this.groupsArray.splice(groupIndex, 1); this.groupsArray.splice(index, 0, group); this.execute(this.CHANGE, group); },

stop

Stops updating and rendering everythin in the requested group. For practical purposes this method pauses and makes invisible all the contents of a group.

Parameters:

  • name must be a String.
    (Id of the group in which to stop all activity)
stop: function(name) { this.stop_update(name); this.stop_draw(name); },

resume

Resumes all activity in the specified group.

Parameters:

  • name must be a String.
    (Id of the group to resume activity in)
resume: function(name) { this.resume_update(name); this.resume_draw(name); },

stop_draw

Stops rendering of the specified group. Effectively making everything in it invisible.

Parameters:

  • name must be a String.
    (Id of the group that should stop rendering)
stop_draw: function(name) { this.groups[name].hide(); },

resume_draw

Resumes rendering of the specified group.

Parameters:

  • name must be a String.
    (Id of the group that should resume rendering)
resume_draw: function(name) { this.groups[name].show(); },

stop_update

Stops updating of the specified group. Effectively pausing everything in it.

Parameters:

  • name must be a String.
    (Id of the group that should stop updating)
stop_update: function(name) { this.groups[name].canUpdate = false; },

resume_update

Resumes updating of the specified group.

Parameters:

  • name must be a String.
    (Id of the group that should resume updating)
resume_update: function(name) { this.groups[name].canUpdate = true; },

allGroupNames

Get an array with all the group names

Returns an Array
(All the group names)

allGroupNames: function() { var r = []; for (var i = 0; i < this.groupsArray.length; i++) { var group = this.groupsArray[i]; for (var k in this.groups) { if (this.groups[k] === group) { r.push(k); } } } return r; },

all

Performs a given action on all the groups. The two arguments are concatenated to form the name of one of the methods described earlier to control groups. ej.:

  • javascript groups.all('stop', 'update');

Would stop the updating of all the groups.

Parameters:

  • action must be a String.
    (This can be either 'resume' or 'stop')

  • method must be a String.
    (This can be either 'draw' or update.)

all: function(action, method) { if (method) action = action + '_' + method; for (var k in this.groups) { this[action](k); } },
