A requireJS module. For use with Game-Builder

This module defines a special kind of game-object-container.

It's only purpose it be used by layers as a way to organize visual elements in a way that makes sense.

It behaives exactly like a game-object-container except for the fact it redefines the clear method so that it only removes all children.
