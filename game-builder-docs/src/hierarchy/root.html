A requireJS module. For use with Game-Builder

This module defines the root game-object-container

In any given Game-Builder application this is is the only game-object-container that is updated explicitly in the main update loop. As it is updated and rendered, it will update and render all of its children, who will in turn update and render their childs, so on and so forth.

If you where wonderin this is where the screen is cleared before each update cycle. It is possible to manipulate this object as any other game-object, but that is probably not a good idea :P

Asides from that, there isn't anything else noteworthy about this module, inspite of being at the very begenning of the hierarchical structure.
