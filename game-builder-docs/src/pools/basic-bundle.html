A requireJS module. For use with Game-Builder

This bundle configures the game-object-pool and the component-pool so that the basic components and game-objects are ready to use.

If you need a configuration to use a simple game-object or a game-object-container there is no need to setup a pool for them, instead you can require this module and use the getGameObjectPoolId and getGameObjectContainerPoolId methods where necessary.

Same goes for renderers if you need ids for a bitmap-renderer, text-renderer or path-renderer you can use the getBitmapRendererPoolId, getTextRendererPoolId and getPathRendererPoolId methods respectively.

Similarly you can get ids for collision-components, either a circle-collider, polygon-collider or fixed-polygon-collider have their corresponding methods. getCircleColliderPoolId, getPolygonColliderPoolId and getFixedPolygonColliderPoolId.

Ej.

var basicBundle = require('basic-bundle');

basicBundle.getGameObjectPoolId() // returns 'game-object'
basicBundle.getGameObjectContainerPoolId() // returns 'game-object-container'
basicBundle.getBitmapRendererPoolId() // returns 'bitmap-renderer'
basicBundle.getTextRendererPoolId() // returns 'text-renderer'
basicBundle.getPathRendererPoolId() // returns 'path-renderer'
basicBundle.getCircleColliderPoolId() // returns 'circle-collider'
basicBundle.getPolygonColliderPoolId() // returns 'polygon-collider'
basicBundle.getFixedPolygonColliderPoolId() // returns 'fixed-polygon-collider'
