A requireJS module. For use with Game-Builder

This is a helper module to define a block of game-object-pool and components-pool configuration.

The basic workflow is extending this module, and putting all the required code to configure the pools in there. That way you can reuse that code in different places.

It's not really needed but it will make things a bit more tidy. Because it has gb as a dependency it already has references to the pools and the canvas, so you don't have to require those in the modules extending this one.
