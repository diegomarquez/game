This demo shows how to capture keyboard input. There is not much explaining to do.

The demo also shows the pause and resume feature of GAME-BUILDER. You can either pause the action manually or in the case the canvas looses focus the action is paused until focus is regained.

Controls

Pressing A Adds a square in a random position in the screen
Pressing S Manual pause
Pressing P Resumes from a manual pause
After adding the maximun amount of objects, the page needs to be refreshed if you want to keep pressing A to add more squares
