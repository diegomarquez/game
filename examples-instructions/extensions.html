This demo shows some extensions in action. The one in charge of pausing the application when it looses focus, the one in charge of resuming the application when it regains focus and another one which resizes the canvas while keeping original aspect ratio if the viewport changes size.

Controls

Clicking in and out the canvas resumes and pauses the application respectively
Resizing the iframe Causes th canvas to resize
Note that clicking on the resize button will make the demo loose focus, pausing it
