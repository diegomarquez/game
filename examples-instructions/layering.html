This demo shows how layering works. Basically you can display stuff in different layers so you don't have to worry about drawing order. Ofcourse there are times when drawing order is important, but layers take care of the bulk of the problem.

The demo sets up three objects that display squares of different colors. If you try to add more than three objects, an error will be shown in console. It doesn't break the demo, but it lets you know that something is up.

Controls

Pressing 1 Adds a red square to the top layer
Pressing 2 Adds a green square to the middle layer
Pressing 3 Adds a blue square to the bottom layer
Pressing A Stops the update of the top layer
Pressing S Stops the rendering of the top layer
Pressing D Resumes all action in the top layer
Pressing Z Clears all the layers
Pressing X Destroys all the layers. This action eliminates all the layers, which means they can not be used any more. The demo needs to be refreshed after doing this.
